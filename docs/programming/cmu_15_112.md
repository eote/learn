Learning CMU 15-112
===================

CMU 15-112 完整详细学习计划（10周版）
------------------------------------

总体目标：夯实Python编程基础，培养计算思维，为后续计算机系统课程打下坚实基础。能够独立开发一个500-1500行代码的综合性Python项目。

#第一阶段：编程基础与核心概念（第1-4周）
目标：掌握Python基础语法、基本数据结构和控制流，理解函数和递归。

##第1周Python入门与环境搭建
###核心内容
*   Python简介与语言特性
*   安装Python、IDE（推荐PyCharm或VSCode）和Git
*   变量、基本数据类型（整数、浮点数、字符串、布尔值）
*   基本输入输出（ print ,  input ）
*   注释与代码风格
###任务与练习
1. 配置开发环境，创建第一个“Hello, World!”程序。
2. 编写程序进行简单的数学运算和字符串拼接。
3. 学习使用Git进行基本代码版本管理（ git add ,  git commit ）。
4. 能够熟练使用IDE编写和运行简单程序，掌握基本的变量操作。


##第2周控制流与逻辑
###核心内容
- 条件语句（ if ,  elif ,  else ）
- 循环语句（ while ,  for ）
- 循环控制（ break ,  continue ）
- 逻辑运算符（ and ,  or ,  not ）
###任务与练习
1. 编写程序判断数字奇偶性、求解最大值/最小值。
2. 使用循环打印九九乘法表、斐波那契数列。
3. 实现一个简单的猜数字游戏。
4. 能够运用条件判断和循环解决基本的逻辑问题。

##第3周函数与递归
###核心内容
- 函数的定义与调用
- 参数传递（位置参数、关键字参数）
- 返回值（ return ）
- 变量的作用域（局部 vs 全局）
- 递归思想与递归函数实现
###任务与练习
1. 将前两周的练习重构为函数形式。
2. 编写函数计算阶乘、幂运算。
3. 使用递归实现汉诺塔问题的求解。
4. 理解函数是代码复用的基本单元，掌握递归的基本思想。

##第4周核心数据结构
###核心内容
- 列表（List）：创建、索引、切片、常用方法（ append ,  pop ,  sort 等）
- 元组（Tuple）：不可变序列
- 字典（Dictionary）：键值对、增删改查- 集合（Set）：无序不重复元素集
###任务与练习
1. 使用列表管理一组数据，并进行排序、筛选等操作。
2. 使用字典构建一个简单的通讯录程序。
3. 利用集合进行去重和集合运算（交集、并集）。
4. 熟练掌握这四种核心数据结构的特性和应用场景。

##第一阶段复盘：
综合运用前四周知识，完成一个小型项目，如“学生成绩管理系统”（支持添加、删除、查询、统计平均分等功能）。

#第二阶段：面向对象与项目实践（第5-7周）
目标：掌握面向对象编程思想，开始使用外部库（如Pygame）进行图形化编程，为最终大项目做准备。

周数学习主题核心内容实操任务与练习学习产出/考核
##第5周面向对象编程（OOP）基础
###核心内容
- 类（Class）与对象（Object）的概念
- 属性（Attributes）与方法（Methods）
- 构造方法（ __init__ ）
- 封装的概念
###任务与练习
1. 定义一个 Student 类，包含姓名、成绩等属性和打印信息的方法。
2. 定义一个 Rectangle 类，包含长、宽属性和计算面积、周长的方法。
3. 能够从“一切皆对象”的角度理解Python，并创建简单的类。

##第6周面向对象编程进阶
###核心内容
- 继承（Inheritance）
- 多态（Polymorphism）
- 模块（Module）和包（Package）的导入与使用
###任务和练习
1. 基于 Student 类，派生一个 GraduateStudent （研究生）类，增加“导师”属性。
2. 学习使用 math ,  random 等标准库模块。
3. 理解OOP的继承和多态特性，能够组织多文件代码。

###第7周图形化编程入门（Pygame）
###核心内容
- Pygame库的安装与窗口创建
- 事件循环（Event Loop）
- 绘制基本图形（矩形、圆形）
- 处理键盘和鼠标事件
###任务和练习
1. 创建一个窗口，并在其中绘制一个随鼠标移动的小球。
2. 实现一个简单的“打砖块”或“贪吃蛇”游戏雏形（仅实现基本移动和绘制）。
3. 掌握图形化程序的基本框架和事件驱动编程模型。


#第三阶段：综合项目与巩固（第8-10周）
目标：整合所学知识，完成一个综合性的课程大项目，并进行代码优化和复盘。

##第8-9周期末大项目攻坚
###核心内容
项目选题（二选一）：    
- 选项A：2D小游戏（如平台跳跃、飞机大战、俄罗斯方块）。    
- 选项B：数据可视化工具（如用图表展示CSV文件中的数据）。

项目开发：    
- 功能拆解与模块化设计。    
- 核心逻辑实现。    
- 用户交互与界面优化。

###任务和练习
全身心投入项目开发。建议采用敏捷开发模式，每周设定小目标并完成。

关键要求：
- 代码量500-1500行。
- 良好的代码结构和注释。
- 使用Git进行版本控制，有清晰的提交记录。

完成一个功能完整、可运行的综合性项目。这是本课程学习成果的核心体现。

##第10周调试、优化与复盘
###核心任务
- 代码调试技巧（使用调试器、打印日志）
- 性能分析与简单优化
- 代码重构与风格检查（如使用Pylint）
- 撰写项目文档（README）
###任务和练习
1. 为自己的项目查找并修复潜在的Bug。
2. 优化代码结构，使其更清晰、易读。
3. 编写项目说明文档，描述功能、安装和运行方式。
4. 回顾整个学习路径，整理知识盲点。
5. 具备基本的代码调试和项目维护能力，形成完整的知识闭环。


#辅助学习建议
- 每日编码：保持每天至少1-2小时的编码习惯，理论结合实践。
- 利用资源：文档中提到的《笨办法学Python》、《Python编程：从入门到实践》等都是极佳的辅助读物。
- 社群交流：如果可能，与同学或在技术社区（如Stack Overflow）交流问题。
- 定期复盘：每周末花时间复习本周内容，并完成综合性的小练习。
